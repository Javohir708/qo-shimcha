Loop

JavaScriptdagi looplar ma'lumotlar to'plamini yoki biror amalni takrorlashda yordam beradi.

JavaScriptda eng ko'p ishlatiladigan looplar quyidagilar:

  1. 1. for - loopi odatda oldindan belgilangan miqdorda takrorlash talab qilinganda ishlatiladi.

    2. for (let i = 0; i < 5; i++) { console.log(i); // 0, 1, 2, 3, 4 }

  2. 2. while - loopi shart true bo'lgan paytda amalni takrorlaydi.

    3. let i = 0; while (i < 5) { console.log(i); i++; }

  3. 3. do while - loopida avval kod bajariladi, keyin shart tekshiriladi.

    4. let i = 0; do { console.log(i); i++; } while (i < 5);

  4. 4. break - shart bajarilganda yoki biror holat yuzaga kelganda siklni darhol tugatishni ta'minlaydi.

    5. for (let i = 0; i < 10; i++) { if (i === 5) { break; // i 5 ga teng bo'lganda sikl to'xtaydi } console.log(i); }

  5. 5. continue - ikl davom etadi, lekin continue dan keyingi kod ishlatilmaydi va keyingi iteratsiyaga o'tiladi.

    6. for (let i = 0; i < 10; i++) { if (i === 5) { continue; // i 5 ga teng bo'lganda o'tkazib yuboradi } console.log(i); }
